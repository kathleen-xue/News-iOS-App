# News iOS App
A simple news app that displays headlines, detailed articles, search trends, and allows users to bookmark news articles they would like to return to.

## APIs Used:
* Guardian News API
* OpenWeatherAPI
* Google Search Trends API
* Bing Autosuggest API

## To Run:
1. Download this repository
1. Open news-ios-app.xcodeproj in Xcode
1. Build with command `command` + `B`, or the â–¶ button in the top left corner of the Xcode UI. This should open up the simulator, which will allow you to experiment with the app! For best results, set iPhone version to iPhone 11 Pro Max.
